## ⚙️ Funcionalidades e Conceitos Aplicado

Durante o desenvolvimento dos exercícios, foram aplicados os seguintes conceitos:

**Instalação do Arquivo:** Inclusão do framework via CDN ou pacotes locais (npm) para ativar o suporte a classes utilitárias e scripts.

**Sistema Grid:** Estrutura baseada em ``container``, ``row`` e ``col`` que utiliza Flexbox para criar layouts responsivos de até 12 colunas.

**Botões:** Classes como ``.btn`` e ``.btn-primary`` que padronizam ações com estilos e estados (hover/active) prontos.

**Formulário:** Estilização de inputs e labels com a classe ``.form-control`` para garantir alinhamento e foco consistentes.

**Cards:** Componente versátil ``.card`` para exibir conteúdos em blocos com bordas, cabeçalhos e rodapés organizados.

**Navbar:** Menu de navegação responsivo que colapsa automaticamente em dispositivos móveis dentro de um "menu hambúrguer".

**Modal:** Janela de diálogo flutuante que bloqueia a interação com a página até que seja fechada ou respondida.

**Carrossel:** Slideshow dinâmico para alternar entre elementos (fotos ou textos) com controles de navegação e indicadores.

**Alerts e Progress:** *Alerts* exibem avisos coloridos contextuais, e *Progress* exibe barras de preenchimento para indicar status de carregamento.
